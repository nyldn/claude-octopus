# /octo:brainstorm

Start a structured brainstorming session — Solo with Claude or Team with multi-AI perspectives.

**Usage:**
```
/octo:brainstorm
/octo:brainstorm [topic]
```

**Modes:**
- **Solo** — Claude-only thought partner session, fast and focused
- **Team** — Multi-AI brainstorm with Codex, Gemini, and Claude for diverse perspectives

**What it does:**
- Solo: Uses Pattern Spotting, Paradox Hunting, Naming the Unnamed, Contrast Creation
- Team: Dispatches parallel queries to available AI providers, synthesizes cross-perspective ideas
- Documents breakthroughs and named concepts

**See:** skill-thought-partner for Solo methodology.

---

**Solo session flow:**
1. Frame the exploration topic
2. Guided questioning (one at a time)
3. Challenge generic claims until specific
4. Collaboratively name discovered concepts
5. Export session with breakthroughs summary

**Team session flow:**
1. Frame the topic
2. Dispatch parallel brainstorm queries to Codex, Gemini, and Claude
3. Collect and present provider-attributed ideas (🔴 🟡 🔵)
4. Synthesize convergence, divergence, and strongest ideas
5. Interactive challenge and building on best ideas
6. Export session with multi-perspective breakthroughs

**Example:**
```
/octo:brainstorm my approach to customer onboarding

→ "How should we brainstorm?" [Solo / Team]
→ (Team selected)
→ 🐙 CLAUDE OCTOPUS ACTIVATED — Multi-AI Brainstorm
→ 🔴 Codex: Technical feasibility and implementation angles
→ 🟡 Gemini: Lateral thinking and ecosystem connections
→ 🔵 Claude: Synthesis, pattern naming, and moderation
```
